# Metabubble
Get out of your stream bubble!

![alt text](https://github.com/stoianmihail/Metabubble/blob/main/public/assets/img/bubble.png?raw=true)

# Setup

We use Firebase:

`
npm install -g firebase-tools
`

# Run

Run the app:

`
firebase serve
`
